 MIRROR — Preventive Health AI

Mirror is an AI-powered wellness system that analyzes facial signals to detect early signs of fatigue, stress, and recovery in real time.

## Features
- Real-time face analysis
- Fatigue / stress / hydration scoring
- AI-generated feedback (Megan)
- Session tracking & comparison

## Tech Stack
- Python (Streamlit)
- Gemini API (Google)
- Computer Vision (face detection)
